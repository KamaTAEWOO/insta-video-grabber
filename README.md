# Insta Video Grabber

공개 인스타그램 **영상 다운로드** + **자막 추출** 크롬 확장 (개인용, Manifest V3).

## 기능
- 📥 **영상 다운로드** — 단일 게시물(`/p/`, `/reel/`)과 **릴스 피드(`/reels/`)**에서 *지금 보고 있는* 영상을 받습니다.
- 📝 **자막 추출** — 영상 화면에 박힌 자막(번인 텍스트)을 읽어 텍스트로 옮깁니다. (OpenAI API 키 필요)

## 설치 (개발자 모드)
1. 크롬에서 `chrome://extensions` 열기
2. 우상단 **개발자 모드** 켜기
3. **압축해제된 확장 프로그램 로드** → 이 폴더 선택
4. 툴바에 아이콘 생성 (📌 고정 권장)

> 코드를 수정하면 `chrome://extensions`에서 **새로고침(↻)** 하고, 인스타그램 탭도 **F5**(content script 재주입)로 새로고침해야 반영됩니다.

## 사용
### 영상 다운로드
1. 인스타그램에서 영상 보기 (단일 게시물 또는 릴스 피드)
2. 영상이 안 잡히면 한 번 **재생**
3. 확장 **아이콘 클릭** → **다운로드** 버튼
4. `instagram_<id>.mp4`로 저장

### 자막 추출 (선택)
1. **설정** 탭에서 OpenAI API 키 입력 (로컬 `chrome.storage`에만 저장, 외부 전송 없음)
2. **자막추출** 탭 → "자막 추출하기"

## 동작 원리
현재 보고 있는 영상을 정확히 집기 위해 여러 단서를 우선순위로 사용합니다.
- **단일 게시물**: 주소(`location.pathname`)의 shortcode로 영상을 정확히 식별.
- **릴스 피드**: 현재 릴스의 shortcode → Instagram 내부 API(`/api/v1/media/{id}/info/`)로 정확한 영상 URL 추출. (페이지에 누적된 '첫 화면' 영상을 집지 않도록 API를 최우선)
- **폴백**: 페이지 내장 JSON 파싱 → `og:video`/`<video>` src → 네트워크 캡처(`PerformanceObserver`).
